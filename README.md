# AlgoLogic 🧠

**Master algorithms from first principles.** 
👉 **Live Demo:** [https://algologic.vercel.app/](https://algologic.vercel.app/)

AlgoLogic is a beautifully designed, interactive, and lightweight single-page web application for exploring essential computer science algorithms and data structures. It provides clear explanations, real-world use cases, interactive visual animations, and runnable implementations in multiple programming languages to help software engineers master algorithmic concepts.

## ✨ Features

- **15 Essential Algorithms**: Explore a curated list of the most critical algorithms across 7 core categories:
  - **Search**: Linear, Binary Search
  - **Sort**: Bubble, Merge, Quick Sort
  - **Graph**: Breadth-First, Depth-First, Dijkstra's, A* Search
  - **Tree**: BST, Trie (Prefix Tree)
  - **Dynamic Programming**: Fibonacci, Knapsack Problem
  - **Greedy**: Activity Selection
  - **Data Structure**: LRU Cache
- **Interactive Visualizations**: Step-by-step visual animations for popular algorithms.
- **Multi-Language Support**: Syntax-highlighted code implementations in **Python**, **JavaScript**, and **Java** for every algorithm.
- **Deep Dives**: Detailed explanations including step-by-step breakdowns, pros & cons, related algorithms, and comprehensive Time & Space Complexity (Big-O) analysis.
- **Light & Dark Mode**: A gorgeous, state-of-the-art UI featuring an intuitive Light and Dark theme toggle that automatically persists your preference via `localStorage`.
- **Zero Dependencies**: Pure HTML, CSS, and Vanilla JavaScript. Absolutely no build scripts or bloated node modules required.

## 🚀 Getting Started

Because AlgoLogic is a zero-dependency architecture, getting started is practically instantaneous:

1. Clone the repository:
   ```bash
   git clone https://github.com/tomoyan/algologic.git
   ```
2. Navigate into the directory and open `index.html` directly in any modern web browser!
   ```bash
   cd algologic
   # On Mac:
   open index.html
   # On Windows:
   start index.html
   # Or simply double-click the file in your file explorer!
   ```

## 🛠️ Built With
- **HTML5** for semantic document structure.
- **Vanilla CSS3** featuring custom properties, modern animations, and dynamic gradient/blur layouts.
- **Vanilla JavaScript (ES6+)** for seamless state management, theme toggling, algorithmic rendering, and visual animation handlers.
- **JetBrains Mono**, **Syne**, and **Instrument Serif** for beautiful, developer-friendly typography.
